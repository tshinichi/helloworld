# 第6回課題
- SpringBootを使って画面にHello World!を表示
- Hello World!を表示したプロジェクトに変更を加える
## やったこと
- @RestController, @RequestParam のアノテーションを使用
- "localhost:8080/hello" の場合、"Hello World!" と出力し、"/hello?name=hoge" の場合、"Hello hoge!"と出力
